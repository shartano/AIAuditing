**Project Title: Computer Vision for Accessibility**
>
> Many organizations struggle to ensure their spaces meet accessibility standards due to costly and time-consuming evaluations, creating barriers in providing inclusive environments. Our project aims to develop an AI/ML-driven solution that automates the detection and auditing of accessibility in public and private spaces like bathrooms. Using Computer Vision, the system will be able to identify key features in
these areas to evaluate the compliance with established accessibility guidelines and provide actionable feedback when standards are not met. In the future, the system could expand to broader contexts such as public transportation, entrances, and other shared environments to ultimately support a scalable, cost-effective approach striving to improve accessibility.
