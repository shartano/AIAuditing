Please refer to the README.md file in the src/prototypes directory for information on how to run the project.

Currently, we do not have a host URL or a compiled/executable code for the project. 
Our current progress is intended to be run in a development environment.

Due to the nature of our project, which involves computer vision and machine learning, we do not have a front facing user interface or application that can be directly accessed via a URL.

We plan to have a React front-end interface in future iterations of the project, which will allow users to interact with the system more easily on web and on mobile devices. 
This interface will be designed to provide a user-friendly experience for evaluating accessibility in various environments.
